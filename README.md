## 🎸 Chi Siamo

Pinakotek è una band strumentale che fonde **electronica, funk psichedelico, ambient e library music** in esperienze sonore ipnotiche e visivamente coinvolgenti.

### Componenti
- **Filippo Cardelli** - Chitarra
- **Nicola Evangelisti** - Basso
- **Luca Follini** - Tastiere
- **Filippo Chiarini** - Batteria
